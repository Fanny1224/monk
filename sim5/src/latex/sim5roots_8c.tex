{\footnotesize\ttfamily long rtbis (\begin{DoxyParamCaption}\item[{double}]{x1,  }\item[{double}]{x2,  }\item[{double}]{xacc,  }\item[{double($\ast$)(double)}]{fx,  }\item[{double $\ast$}]{result }\end{DoxyParamCaption})}

Root finding. Finds root of a function on an interval. Using bisection method, finds the root of a function {\ttfamily fx} that is known to lie between {\ttfamily x1} and {\ttfamily x2}. The root, returned as {\ttfamily result}, will be refined until its accuracy is +/-\/ xacc.

\hline

\hline
{\em fx} & function\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Returns 1 if OK and the root position in {\ttfamily result}, 0 if error.
